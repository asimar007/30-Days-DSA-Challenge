# Sum of Two Integers

Problem link: [Sum of Two Integers - LeetCode](https://leetcode.com/problems/sum-of-two-integers/description/)

## Problem Description

Given two integers `a` and `b`, return the sum of the two integers without using the operators `+` and `-`.

## Example

Input: `a = 1, b = 2 `
Output: `3`

Input: `a = -2, b = 3`  
Output: `1`
